# Lazuli Premium Ready

## Instalação

```bash
npm install
npm run dev
```

Abra: http://localhost:3000

## Onde trocar

- Logo SVG: `public/logo/lazuli-logo.svg`
- Hero imagens: `public/images/`
- Foto da Aline: `public/professionals/aline-reis.jpg`
- Links e números: `data/site-data.ts`

## WhatsApp por profissional

O botão `Confirmar e abrir WhatsApp` no modal usa o número da profissional escolhida.
Troque os números em `data/site-data.ts`.
